
# EscapeRoomTim

Dieses Repository enthält einen Linux Escape Room mit mehreren Benutzern und Rätselordnern.

## Installation und Start

1. Repository klonen:
   git clone https://github.com/Tim12847/EscapeRoomTim.git
   cd EscapeRoomTim

2. Setup-Skript ausführen (legt Benutzer, Ordner und Rechte an):
   sudo ./setup.sh

3. Spielstart:
   Wechsle in das Home-Verzeichnis und lies die Datei `start.txt`.  
   Dort stehen alle weiteren Hinweise zum Ablauf des Escape Rooms.
